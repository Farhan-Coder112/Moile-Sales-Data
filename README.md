# Moile-Sales-Data
Recommended Structure and Order

Project Title.
Mobile Sales Performance Analytics Dashboard from year 2021-24.

Short Description.
This Power BI project provides an interactive and visually rich dashboard to analyze mobile sales performance across multiple dimensions.
The purpose is to help businesses track total sales, quantity sold, transactions, monthly trends, brand-wise performance, payment methods, ratings, and year-over-year comparisons.
It enables data-driven decision-making with clear insights into market trends, customer behavior, and product performance.

Tech Stack.
-Power BI Desktop.
-Power Query.
-DAX (Data Analysis Expressions).
-Excel / CSV for Data Modeling.
-Power BI Visualization Tools.
-Map Visualization (Filled Map & Bubble Map).

Data Source.
-CSV/Excel files containing:
-Mobile brand & model sales data
-Quantity and transaction logs
-Monthly/quarterly/yearly sales records
-Payment method details
-Rating and customer feedback data
-Geographic location-wise sales
-Transformed and cleaned using Power Query before loading into the data model.
